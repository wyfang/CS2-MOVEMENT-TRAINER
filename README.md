## 🌟 Enhanced Features / 本版本新增功能
- 🌐 **Simplified Chinese Support**: Full localization of UI, menus, and statistics. (完整简体中文汉化)
- ☀️ **Light Mode**: Added a light theme for better visibility in different environments. (新增亮色主题模式)

## 原项目介绍 / Original Project
Counter-Strike 2 counter-strafe trainer built with PIXI.js and a Data-Oriented Design (DOD) architecture. This version is a refactoring of the original trainer, optimized for rendering efficiency and modular logic.
